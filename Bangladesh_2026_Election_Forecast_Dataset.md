## 13. Model Methodology Summary

| Component | Approach |
|---|---|
| Data inputs | 7 national surveys; 80,000+ aggregated respondents |
| Historical calibration | 30-year FPTP seat-conversion ratios (1991–2024) |
| AL voter redistribution model | CRF/BEPOS migration data applied to 2024 AL vote distribution |
| Rebel candidate adjustment | −15 to −25 BNP seats (79 affected constituencies) |
| Undecided allocation | Split 55% BNP / 30% Jamaat / 15% others (based on late-break historical patterns) |
| Turnout elasticity | +1 pp turnout ≈ +2–3 BNP seats (BNP base more turnout-elastic) |
| Gen Z correction | +10 to +20 Jamaat seats (campus sweep extrapolation) |
| Geographic weighting | Rural BNP advantage × urban Jamaat/NCP concentration |
| Confidence intervals | ±30 seats for BNP; ±27.5 seats for Jamaat alliance |
